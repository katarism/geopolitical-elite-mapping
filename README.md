# 地缘政治精英网络图谱工具 (Geopolitical Elite Mapping)

> 🟢 **[2026-03-28 更新] 🚀 重磅功能：可视化编辑器 (Visual Editor) 已上线！**  
> 现在的工具不仅能看，还能直接“上手”修改。您可以在浏览器中直接增删节点、调整关系，完全不需要编写代码。

这是一个专为研究人员设计的地缘政治关系制图包。它能将复杂的精英人物、组织及其背后的利益纽带，转化为直观、可交互的关系图谱，帮助您在层层迷雾中识破权力网络。

### 🌟 它能为您做什么？

- **一眼看透权力格局**: 自动识别“核心人物”、“阵营划分”及“利益节点”。
- **跨越时代的分析**: 支持查阅不同历史时期的关系演变（如“冷战期” vs. “现今”）。
- **信息可靠性标注**: 通过视觉样式（实线、虚线、粗细）直接体现信息的可靠程度。
- **全平台离线运行**: 数据保存在您的私有仓库中，无需联网即可在浏览器中展示和分析。

---

## 🎨 可视化编辑器：像画图一样简单

进入 `Editor/` 目录运行 `editor.html`，即可开启您的创作。

- **右键即刻编辑**: 在空白处右键新增人物，在人物上右键修改他的阵营、备注或头像。
- **直观的连线管理**: 无需在细线上费力点击。通过侧边栏的“搜索+下拉”框，轻松建立两个人物之间的复杂联系。
- **自动阵营吸附**: 随意拖拽，系统会自动帮您划入某个势力范围（阵营框），逻辑关系自动计算。
- **一键导出成果**: 随时将您的研究成果导出为高清图片（PNG/SVG）或用于存储的数据文件。

### 🚀 如何快速开始？

1. **准备数据**: 运行主目录下的脚本（或使用现有 `data.json`）作为起步底图。
2. **配置您的 GitHub 仓库 (安全云储存)**:  
   为了防止数据丢失并支持多端同步，编辑器会将您的修改安全地存储在 GitHub 的私有仓库中。
   - **获取授权码 (Token)**: 前往 GitHub 设置申请一个 `Fine-grained Token`。
   - **安全建议**: 在配置时仅勾选 `Only select repositories`（指定您的数据仓库），并在权限中仅勾选 `Contents (Read and Write)`。这能确保即使 Token 泄露，您的其他代码也绝对安全。
3. **打开并工作**: 直接双击 `Editor/editor.html`，输入授权码和仓库名，导入数据即可开工！

## 🛠️ 环境与安装 (技术小贴士)

如果您是研究人员，只需简单几步即可搭建环境：

1. **Python**: 安装 3.10 或更高版本。
2. **Graphviz**: 系统需安装绘图引擎（macOS 用户执行 `brew install graphviz` 即可）。
3. **获取本项目**:  
   ```bash
   git clone https://github.com/katarism/geopolitical-elite-mapping.git
   cd geopolitical-elite-mapping
   ```
4. **生成首张图谱**:  
   执行 `python3 generate_elite_network.py`，您的第一张交互式图谱就生成在当前目录下了。
